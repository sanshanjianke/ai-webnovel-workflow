## 下载脚本模板

```bash
#!/bin/bash
# 保存为 download_arxiv.sh
# 使用: ./download_arxiv.sh arxiv_id1 arxiv_id2 ...

for id in "$@"; do
    echo "Downloading $id..."
    wget -q --show-progress "https://arxiv.org/pdf/$id.pdf" -O "${id}.pdf"
    sleep 1
done
```
